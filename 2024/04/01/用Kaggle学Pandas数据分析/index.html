前言：Pandas 总是查了一点点文档就用在工作里，几乎没有系统学过。刚好Kaggle有这么个教学，一边联系一边记录吧。

Create a new DataFrame

standard usage

pd.DataFrame({‘Yes’: [50, 21], ‘No’: [131, 2]})

  • with index parameter,pd.Series also have index parameter.

pd.DataFrame({‘Bob’: [‘I liked it.’, ‘It was awful.’],
‘Sue’: [‘Pretty good.’, ‘Bland.’]},
index=[‘Product A’, ‘Product B’])

pd.Series([30, 35, 40], index=[‘2015 Sales’, ‘2016 Sales’, ‘2017 Sales’], name=’Product A’)

  • use shape to check/summary the DF’s weigth and heigth:

df.shape

read DataFrames:

  • read a col (example as a book ) by col’s name

useage1:

books.title

books.author

etc…

usage2:

book[‘title’]

books[[‘title’,’author’]]

  • read by idx of rows and cols (rows first and cols second)

df.iloc[rows range,cols range],e.g.:

books.iloc[4] #use for read row4 of books DataFrame

books.iloc[:,4] #use for read entire

df.iloc[[1,2,-12],8] # select by index at from head count the 1\2\rows,and from bottom count the 12 row

  • read by label of row index and col lebels:

reviews.loc[0, ‘country’]

reviews.loc[:, [‘taster_name’, ‘taster_twitter_handle’, ‘points’]]

  • iloc and loc

iloc:number,loc:label

iloc: [), loc:[]

operation of dataframes

  • selection by conditions

return a boolean series e.g.:

reviews.country == ‘Italy’

input this series to .loc[], can see instantantly the selected table

reviews.loc[(reviews.country == ‘Italy’) & (reviews.points >= 90)]

  • conditional selectors

mutiple conditions can use | or & to build complex selectional conditions

isin: reviews.loc[reviews.country.isin([‘Italy’, ‘France’])]

isnull/isnotnull: reviews.loc[reviews.price.notnull()]

  • creat a new col

reviews[‘index_backwards’] = range(len(reviews), 0, -1)

summary function and maps

  • summary functions :

reviews.points.describe()#count\mean\max…for number col,count\unique\top for string col

reviews.points.mean()

reviews.taster_name.unique()

reviews.taster_name.value_counts()

  • maps （映射）:return a new object,instead of change origin col/row

map:

二值化、归一化……

apply:

call a custom method on each row

.apply(CUSTOM METHOD,AXIS)

AXIS: columns\rows
